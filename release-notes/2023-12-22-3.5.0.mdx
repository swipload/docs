## Freight orders

### Features

#### Consignor & Consignee reference available in /freight (API) endpoint as input from external systems. #2956

### Fixes

- Showing actual pickup and delivery date/time on the POD. #2996
- Cancelling a freight order in some cases showed something went wrong though it didn't. #2946
